# Smartbuildes - Construction Management System 🏗️

Smartbuildes is a full-stack Django web application designed to help users and construction companies manage home building projects, estimate material costs, and generate professional invoices.

## 🚀 Key Features
* **Interactive Home Page:** Modern UI with dynamic sliders and brand carousels.
* **Construction Cost Calculator:** Real-time estimation of building costs based on location (Metro, Urban, etc.) and square footage.
* **Dynamic Material Pricing:** Cascading dropdowns to check real-time prices for Civil, Electrical, and Plumbing materials.
* **Project Management:** Add and track ongoing construction projects.
* **Material Tracking:** Add specific materials and quantities to individual projects.
* **Automated Billing:** Generate highly professional, printable PDF-style invoices with automatic labor cost and profit margin calculations.

## 💻 Tech Stack
* **Backend:** Python, Django
* **Frontend:** HTML5, CSS3, JavaScript, Swiper.js
* **Database:** SQLite (Development)
* **Styling:** Custom CSS with Premium UI/UX design principles and FontAwesome icons.
